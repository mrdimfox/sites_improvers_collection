How to install modifiers
========================

Scripts
-------

For almost all scripts you can use :extension:`Tampermonkey` extension for
your browser. Here are some links to extensions:

* `Firefox <https://www.tampermonkey.net/?ext=dhdg&browser=firefox>`_;
* `Chrome-based browsers <https://www.tampermonkey.net/?ext=dhdg&browser=chrome>`_;
* `Edge <https://www.tampermonkey.net/?ext=dhdg&browser=edge>`_  (not Chrome-based version);
* `others <https://www.tampermonkey.net/index.php?ext=dhdg&browser=safari>`_.
