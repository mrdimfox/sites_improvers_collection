How to install modifiers
========================

Scripts
-------

For almost all scripts you can use :extension:`Tampermonkey` extension
for your browser. Here are some links to extensions:

* `Firefox <https://www.tampermonkey.net/?ext=dhdg&browser=firefox>`_;
* `Chrome-based browsers <https://www.tampermonkey.net/?ext=dhdg&browser=chrome>`_;
* `Edge <https://www.tampermonkey.net/?ext=dhdg&browser=edge>`_  (not Chrome-based version);
* `others <https://www.tampermonkey.net/index.php?ext=dhdg&browser=safari>`_.

Right after extension installation you can install any of scripts:

1. Open one of the script files in Github `like that <https://github.com/
   mrdimfox/sites_improvers_collection/blob/master/tampermonkey/easy_redmine/
   easy_redmine_toc.js>`_.

2. Then press :guilabel:`Raw` button. Copy an url of the page into a clipboard.

3. Open :extension:`Tampermonkey` extension :menuselection:`Dashboard -->
   Utilites`.

4. Paste the url into *Install from URL* field and press :guilabel:`Install`
   button.

5. Press another :guilabel:`Install` button.

6. Your are DONE!

Script will be enabled automatically on a page with url which matches
the regexp in `@include` section of the script header.
